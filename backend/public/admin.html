💡 What Each Action Means

Action What It Does
Replace All Items Deletes old master items, then inserts CSV as a fresh master list.
Merge (Add/Update) Keeps existing items and only adds new barcodes or updates changed rows.
Push Master Data Makes phones download the latest item master version on the next manual sync.
Processed / Archived Queue Keeps history for audit and KPI reporting without sending old work back to phones.
