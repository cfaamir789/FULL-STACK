Action What It Does

📚 ERP Queue — How to Process
1
💾 Take Backup
Go to Backup & Restore and take a full backup of the current queue. Do this before marking anything done.
2
⬇️ Download Backup
Download the backup file to your computer. Keep it safe — this is your safety net if anything goes wrong.
3
✓ Select Workers & Mark Done
Filter by one or more workers below. Review the rows, then use Mark Worker Done or Mark Filtered Done to move them to history.
4
📷 Clear Phone
Once marked done, the worker can clear their phone queue on next sync. Workers will see 0 pending after their app syncs.
⚠️ Important: Always take & download the backup first (Steps 1–2) before marking rows as done. The Date column copies without time — use it when pasting into ERP.
